A FaceBook Clone implemented using Next-JS, Java SpringBoot and Redux

1. Create-Next-App + TailWindCSS to setup frontend application
2. Implemented a Login page using Next-Auth and FaceBook Provider
3. Facebook-like Interface for the UI
4. Axios to handle frontend API calls
5. Redux and Redux Toolkit to manage state of application and manage session
6. Built a Backend Service using Java SpringBoot
7. REST API controller, service and repository layers added
8. SpringBoot application built using Maven, JDBC, MySQL Database
9. Post and Get Mapping for Posts that can be added to User's Feed
10. All Data visible on frontend and in backend MySQL Database instance


Software Used :
1. IntelliJ IDEA 2024.1.4 (Ultimate Edition) for Java Spring Boot using Maven
2. VSCode (Community Edition) for Next-JS app using CNA
3. DataGrip (By JetBrains) to manage DB instance
